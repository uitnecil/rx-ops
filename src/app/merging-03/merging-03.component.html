How to use:
  • Map (static method/ instance method): Offers the possibility for each value of the stream to be altered based on a function.
  • Concat (static method/ instance method): Concatenates the output of two streams. Values from first stream are returned first and once it is completed, the second stream values are returned. If first stream is infinite, the values from the second stream will never be emitted.
  • Merge (static method/ instance method): Merges the values between two streams. Values are returned individually in the order they were emitted by each stream.
