How to use:
  • Map: static method/ instance method.
  • Concat (static method/ instance method): Concatenates two streams. If first stream is infinite, the values from the second stream will never be emitted.
  • Merge: static method/ instance method
  • .map() Description: Apply projection with each value from source.
    signature: map(project: Function, thisArg: any): Observable
  • .merge() Description: Turn multiple observables into a single observable.
    signature: merge(input: Observable): Observable
  • .concat() Description: Subscribe to observables in order as previous completes, emit values.
    signature: concat(observables: ...*): Observable
