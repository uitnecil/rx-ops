Operator Details

  • .combineLatest() Description: When any observable emits a value, emit the latest value from each.
    signature: combineLatest(observables: ...Observable, project: function): Observable
  • .zip() Description: After all observables emit, emit all values with the same index as an array
    signature: zip(observables: *): Observable
  • .forkJoin() Description: When all observables complete, emit the last emitted value from each.
    signature: forkJoin(...args, selector : function): Observable
  • .race(Obs1, Obs2) Description: Returns the observable to emit first.
    signature: race(observables: ...Observable): Observable
How to use:
  • combineLatest (static method/ instance method): Returns the combination of the latest values from each stream. Values starts once both streams have returned at least one individual value.
  • zip (static method/ instance method): Returns the combination (array) of the values having the same index, from each stream (both streams have returned values corresponding to the same index). If only one stream emits no results is emitted.
  • forkJoin (static method): Returns an array containing the last values returned by the individual observables before completion (doesn't matter how many values have been returned before completion).
  • race (static method/ instance method): Uses the values from the first emitting stream. If the first stream emits an error, the error is used and the other streams are ignored.
